# Urdu Spell Corrector

Yeh ek Urdu spell checker hai jo Minimum Edit Distance ka use karta hai.

### Features
- Urdu text ki spelling mistakes correct karta hai
- Common mistakes jaise "مین" → "میں", "نی" → "نے" handle karta hai
- Gradio interface ke saath

### How to use
Textbox mein Urdu text daalo aur "Check کریں" button dabaao.
